Fundamentos do JavaScript

O JavaScript é uma linguagem de programação usada principalmente para criar interatividade em páginas web. Ele é uma linguagem de alto nível, interpretada e orientada a objetos, essencial para o desenvolvimento front-end, mas também pode ser usado no back-end (ex: Node.js). Os fundamentos do JavaScript envolvem o uso de variáveis, tipos de dados, operadores, estrutura de controle, funções e manipulação do DOM(Documnet Object Model), que é a representação da página web no código.

Variáveis e Dados

Em JavaScript, variáveis são usadas para armazenar dados. Você pode declarar variáveis usando as palavras-chaves var, let ou const. A diferença entre elas está no escopo e mutabilidade da variável.

  • let permite a reatribuição de valores e tem escopo de bloco.
  • const é usada para declarar valores constantes(não podem ser reatribuídos)
  • var tem um escopo global ou de função e é menos recomendada hoje em dia.
Os principais tipos de dados em JavaScript são:
  • Primitivos: number, string, boolean, idefined, null, symbol, biginy.
  • Não primiticos: object, que inclui arrays, funções, e objetos

Operadores

Os operadores em JavaScript são usados para realizar operações sobre valores. Exitem vários tipos de operadores:

  • Aritméticos: +, -, *, /, % (para soma, subtração, multiplicação, divisão, e resto).
  • Relacionais: ==, ===, != !==, >, <, >=, <= (para comparar valores).
  • lógicos: &&, ||, ! para operações booleanas.
  • Atribuição: =, +=, -=, *=, /=, etc
  • Ternário: Um operador condicional (condição? valor_se_verdadeiro : valor_se_falos).

Condições

As Condições permitem que o código execulte diferentes blocos de código com em condiçoes específicas. em JavaScript, as principais estruturas de controle condicional são:

  • if: Executa um bloco de código se a condição for veradeira.
  • else: Executa outro bloco de código se acondição if for falsa
  • else if: Verifica várias condições sequenciais.
  • switch: Uma alternativa ao if quando se tem várias condições a verifiacar, especialmente para comparar um único valor com múltiplos casos.

Repetições

As estruturas de Repetições permitem executar um bloco de código várias vezes. As principais são:

  • for: Executa um bloco de código enquanto uma condição for verdadeira.
  • while: Executa um bloco de código enquanto a condição for verdadeira.
  • do...while: Executa o bloco de código pelo menos uma vez, antes de verificar a condição.

Arrays

Arrays são coleções ordenadas de dados, onde podemos armazenar múltiplos valores em uma única variável. Eles podem conter qualquer tipo de dado (números, strings, objetos, etc.). Métodos comuns para manipular arrays incluem push() (adicionar), pop() (remover), shift() (remover o primeiro), unshift() (adicionar no início), entre outros.

Funções

Funções são blocos de códigos reutilizáveis. elas podem receber parâmetros(dados) e retornar um valor. Existem duas formas principais de declarar funções em JavaScript

  • Declaração de função
  • Função anônima (ou arrow function)
Funções podem ser chamadas em qualquer lugar do código e ajudam a manter o código organizado e reutilizável.

DOM (Document Object Model)

O DOM (Document Object Model) é uma interface de programação que permite manipular a estrutura de um documento HTML ou XML em JavaScript. Ele permite acessar e modificar os elementos da página web dinamicamente.

  • Acessar elementos: document.getElementById(), document.getElementsByClassName(), document.querySelector().
  • Modificar conteúdo: element.innerHTML, element.textContent.
  • Alterar atributos: element.setAttribute(), element.getAttribute().
  • Adicionar eventos: element.addEventListener(), como click, keydown, etc.

Desafio: Hora do Dia

Descrição: Desenvolvi um pequeno projeto que exibe a hora do dia e muda a imagem e o fundo da página com base no horário atual . Ao carregar a página, ela verifica a hora atual e ajusta o conteúdo e o estilo da página conforme a hora do dia: manhã, tarde ou noite.

Implementação: O projeto foi implementado usando HTML, CSS e JavaScript. Utilizei JavaScript para obter a hora atual e, com base nisso, alterei a imagem exibida e a cor de fundo da página. A estrutura HTML define os elementos da página, e o CSS cuida da estilização.

Aprendizado: Com este desafio, aprendi a manipular o DOM (Document Object Model) com JavaScript para alterar dinamicamente o conteúdo da página . Além disso, pratiquei a lógica condicional para verificar a hora do dia e aplicar as mudanças apropriadas.

Desafio: Verificador de Idade

Descrição: Desenvolvi um projeto que verifica a idade de uma pessoa com base no ano de nascimento inserido e no gênero selecionado. Dependendo da idade, a página exibe uma imagem correspondente (criança, jovem, adulto ou idoso) e uma mensagem com o resultado.

Implementação: Utilizei HTML para estruturar a página, CSS para estilização e JavaScript para implementar a lógica de verificação e manipulação do DOM. O JavaScript coleta os valores inseridos pelo usuário, calcula a idade e exibe o resultado juntamente com uma imagem apropriada.

Aprendizado: Com este desafio, aprendi a usar JavaScript para manipular o DOM e responder aos eventos do usuário. Também pratiquei lógica condicional para determinar e exibir a imagem correta com base na idade calculada.

Desafio: Vamos Contar

Descrição: Desenvolvi um projeto que conta números de um valor inicial até um valor final, com incrementos definidos pelo usuário. A página exibe a contagem progressiva dos números de acordo com os parâmetros inseridos (início, fim e passo).

Implementação: Utilizei HTML para estruturar a página, CSS para a estilização e JavaScript para implementar a lógica de contagem e manipulação do DOM. A lógica de contagem verifica os valores inseridos pelo usuário e exibe a sequência de números na tela.

Aprendizado: Com este desafio, aprendi a usar JavaScript para manipular o DOM, coletar dados de entradas do usuário e implementar loops para gerar uma sequência de números. Também aprendi a validar entradas do usuário para garantir que os valores são apropriados para a contagem.

Desafio: Tabuada

Descrição: Desenvolvi um projeto que gera a tabuada de um número inserido pelo usuário. A página exibe a multiplicação do número escolhido de 1 a 10.

Implementação: Utilizei HTML para estruturar a página, CSS para estilização e JavaScript para implementar a lógica de geração da tabuada. O JavaScript coleta o valor inserido pelo usuário, gera a tabuada e exibe os resultados na tela.

Aprendizado: Com este desafio, aprendi a usar JavaScript para manipular o DOM, coletar dados de entradas do usuário e implementar loops para gerar uma sequência de multiplicações. Também aprendi a validar entradas do usuário para garantir que os valores são apropriados para a operação desejada.

Desafio: Analisando Números

Descrição: Desenvolvi um projeto que permite ao usuário adicionar números entre 1 e 100 a uma lista e, após finalizar a entrada dos números, exibe uma análise dos dados inseridos. A análise inclui a quantidade de números, o maior e o menor valor, a soma de todos os números e a média dos valores digitados.

Implementação: Utilizei HTML para estruturar a página, CSS para estilização e JavaScript para implementar a lógica de coleta e análise dos números. O JavaScript coleta os valores inseridos pelo usuário, armazena-os em uma lista, e realiza os cálculos necessários para exibir a análise final.

Aprendizado: Com este desafio, aprendi a usar JavaScript para manipular o DOM, validar entradas do usuário e realizar cálculos com base nos dados coletados. Também pratiquei a criação de elementos dinâmicos na página para exibir os resultados da análise.
